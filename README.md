## Постановка задача 
Проведите анализ разнородных геоданных о зданиях и предложите подход к построению единой модели высотности, пригодной для задач планирования
сети. Для этого:
  1. Проанализируйте качество данных, предоставленных из источников, и предложите методы очистки.
  2. Разработайте способ сопоставления объектов, относящихся к одному физическому зданию, в условиях, когда один дом в разных источниках может
  быть представлен по-разному.
  3. Предложите алгоритм или модель определения наиболее достоверной высоты здания и обоснуйте свой выбор.
     
   Если ни один из источников не содержит информации о высоте здания, предложите алгоритм ее оценки по косвенным признакам (например, на
    основе площади застройки, назначения здания, этажности соседних зданий, данных геокодера или иных открытых данных). Обоснуйте примени-
    мость предложенного подхода.

  5. Проведите валидацию полученных результатов.

  6. Визуализируйте итоговый слой зданий на карте и сформулируйте инсайты о структуре застройки города, которые могут быть полезны для планирова-
ния сети.

## ОПИСАНИЕ ПРЕДОСТАВЛЕННЫХ ДАННЫХ
### ИСТОЧНИК А                                                                              
Размер файла: 39 Мб.
Количество записей: 171 453.

Поля:

  id — идентификатор полигона;
  
  title — наименование объекта;
  
  tags — теги (тип объекта);
  
  geometry — геометрия в формате WKT;
  
  area_sq_m — площадь полигона, м2;
  
  gkh_address — адрес из базы «Реформа ЖКХ»;
  
  gkh_floor_count_min — минимальная этажность по данным приложения «Реформа ЖКХ»;
  
  gkh_floor_count_max — максимальная этажность по данным приложения «Реформа ЖКХ».

### ИСТОЧНИК Б
Размер файла: 82 Мб.
Количество записей: 160 610.

Поля:

  subject — субъект РФ;
  
  district — муниципальный район;
  
  type — тип населенного пункта;
  
  locality — наименование населенного пункта;
  
  type_street — тип улицы;
  
  name_street — наименование улицы;
  
  number — номер дома;
  
  letter — литера;
  
  fraction — дробь;
  
  housing — корпус;
  
  building — строение;
  
  purpose_of_building — назначение здания;
  
  stairs — этажность;
  
  avg_floor_height — средняя высота этажа, м;
  
  height — высота здания, м;
  
  wkt — геометрия в формате WKT.
